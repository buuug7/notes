# licences 许可证

## 常用 licences

#### Apache License 2.0

主要条件需要保留原有版权和许可证通知，对修改的代码需要提供申明变更。许可作品，修改和大型作品可以在不同的条款下分发，无需源代码。

- 要求
  - 协议和版权信息
  - 申明变更
- 允许
  - 商用
  - 分发
  - 修改
  - 专利授权
  - 私用
  - 附加协议
- 禁止
  - 责任承担
  - 商标使用
